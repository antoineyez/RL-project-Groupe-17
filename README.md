# Projet Reinforcement Learning - Groupe 17

Ce dépôt contient les travaux réalisés dans le cadre du projet de Reinforcement Learning (Mention IA). L'objectif est d'entraîner et d'analyser des agents de RL dans l'environnement `highway-env`.

## Structure du Projet

* `core/` : Implémentation du DQN "maison" et scripts d'entraînement SB3.
* `extension/` : Code relatif à la tâche d'extension choisie par le groupe.
* `configs/` : Fichiers de configuration (dont `shared_core_config.py`).
* `results/` : Courbes d'apprentissage, checkpoints et vidéos des agents.
* `scripts/` : Utilitaires pour l'évaluation et la visualisation.
